# Sales Data Analysis using Python

##  Project Overview
This project analyzes sales data using Python and Pandas to extract useful business insights.

##  Features
- Calculate total sales revenue
- Find best-selling product
- Analyze sales by city and category
- Sort products based on quantity sold

## Tools Used
- Python
- Pandas

##  Dataset
The dataset contains order details like product, category, price, quantity, and city.

##  Key Learning
This project helped me understand:
- Data analysis using Pandas
- GroupBy operations
- Sorting and filtering data
